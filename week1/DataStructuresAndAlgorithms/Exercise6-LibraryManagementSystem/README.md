# Exercise 6 - Library Management System

## Objective

Implement Linear Search and Binary Search for searching books by title.

## Algorithms Used

- Linear Search
- Binary Search

## Time Complexity

### Linear Search
- Best Case: O(1)
- Average Case: O(n)
- Worst Case: O(n)

### Binary Search
- Best Case: O(1)
- Average Case: O(log n)
- Worst Case: O(log n)

## Conclusion

Binary Search is more efficient for large sorted datasets.
