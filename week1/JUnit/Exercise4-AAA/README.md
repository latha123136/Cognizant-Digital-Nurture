# Exercise 4 - Arrange Act Assert (AAA)

## Objective
Learn the Arrange–Act–Assert (AAA) testing pattern for writing clean and maintainable unit tests.

## AAA Pattern

### Arrange
Create the required objects and initialize test data.

### Act
Invoke the method that needs to be tested.

### Assert
Verify that the actual result matches the expected result.

## Conclusion
Successfully implemented JUnit test cases using the Arrange–Act–Assert testing pattern.
