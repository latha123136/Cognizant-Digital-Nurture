# Exercise 2 - E-commerce Search

## Objective

Implement Linear Search and Binary Search.

## Algorithms

- Linear Search
- Binary Search

## Time Complexity

### Linear Search

Best Case : O(1)

Average Case : O(n)

Worst Case : O(n)

### Binary Search

Best Case : O(1)

Average Case : O(log n)

Worst Case : O(log n)
