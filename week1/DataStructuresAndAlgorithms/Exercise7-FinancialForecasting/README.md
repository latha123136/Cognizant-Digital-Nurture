# Exercise 7 - Financial Forecasting

## Objective

Predict future financial values using Recursion.

## Formula

Future Value = Present Value × (1 + Growth Rate)^Years

## Algorithm Used

Recursive Algorithm

## Time Complexity

O(n)

## Space Complexity

O(n)

## Optimization

The recursive solution can be optimized using:

- Memoization
- Dynamic Programming
- Iterative Approach
